## Technologies Used

- **HTML5** - Semantic tags, W3C standards compliant
- **CSS3** - Responsive design with CSS variables and media queries
- **JavaScript / jQuery 3.7.1** - Form validation and interactive animations
- **Google Fonts** - Oswald & Open Sans

## Features

### Form Validation (jQuery)
- Login form: email + password validation
- Register form: email, password strength (6-20 chars), password match
- Forgot password: email format validation
- Contact form: all required fields validation
- Payment form: card number (16 digits), CVV (3-4 digits), zip code, terms acceptance
- Edit forms: all profile & address forms validated

### Interactive Animations
- Hero slider with auto-play, dots and arrow navigation
- Product image thumbnail switcher
- Tabbed product details (Details, Specs, Accessories, Reviews, Q&A)
- Related products carousel with prev/next navigation
- Dynamic price calculator on product detail page
- Add to cart animation (color feedback)
- Shipping address toggle (show/hide)
- Cart item removal with fade animation
- Cart total auto-recalculation
- Page load content fade-in animation
- Back-to-top button (scroll-triggered)
- Search bar focus expand animation

### Responsive Design
- Mobile-first media queries at 768px and 480px
- Fluid grid layouts
- Collapsible mobile navigation
