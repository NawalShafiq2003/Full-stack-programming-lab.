Overview

jQuery is a fast, small, and feature-rich JavaScript library. It simplifies HTML document traversal and manipulation, event handling, CSS animation, and Ajax with an easy-to-use API that works across browsers.

Originally released in 2006 by John Resig, jQuery became the most widely used JavaScript library on the web, powering millions of websites and web applications.

Release Year2006
CreatorJohn Resig

Features

jQuery provides a comprehensive set of features for modern web development. The library abstracts away browser inconsistencies so you can focus on writing clean, concise code.

  • DOM traversal and manipulation using CSS-style selectors
  • Cross-browser event handling with .on() and shorthand methods
  • Built-in effects: hide, show, fade, slide, and animate
  • CSS manipulation via .css(), .addClass(), .removeClass()
  • Method chaining to apply multiple operations in one statement
  • Simplified Ajax requests with $.get(), $.post(), and $.ajax()
  • Plugin ecosystem with thousands of community extensions

jQuery Concepts

The core concept of jQuery is the selector-action pattern. You select an element and then perform an action on it. This is done through the $() function.

DOM Manipulation

Select and modify elements dynamically. Methods include .text(), .html(), .val(), .attr(), .append(), and .remove().

Event Handling

Attach event listeners to user interactions. The .on() method handles click, hover, keypress, submit, and more.

Effects and Animations

Animate elements smoothly using .fadeIn(), .fadeOut(), .slideToggle(), and .animate() for custom transitions.

Usage

To include jQuery in your project, add the CDN link inside the head or just before the closing body tag:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Then wrap all jQuery code inside the document-ready callback to ensure the DOM is fully loaded before executing:

$(document).ready(function () {
  // Your jQuery code here
  $("button").on("click", function () {
    $("p").fadeToggle(400);
  });
});
