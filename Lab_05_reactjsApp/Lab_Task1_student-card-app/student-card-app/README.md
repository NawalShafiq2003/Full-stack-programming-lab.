# Lab Task 1: Student Information Card App

**Course:** Full Stack Programming — Lab 05 (Frontend React JS)  
**Instructor:** Mr. Sharif Hussain  
**Class:** BSSE-VI-B & A

## Objective
Practice creating React components and passing data using props.

## Features
- `StudentCard` reusable functional component
- Props: `name`, `rollNo`, `department`, `university`, `color` (bonus)
- 3 student cards rendered dynamically from an array using `.map()`
- Hover animations with smooth transitions

## How to Run

```bash
npm install
npm start
```

Then open http://localhost:3000 in your browser.
