# Lab Task 3: Dynamic Greeting App

**Course:** Full Stack Programming — Lab 05 (Frontend React JS)  
**Instructor:** Mr. Sharif Hussain  
**Class:** BSSE-VI-B & A

## Objective
Use props with conditional rendering based on the `timeOfDay` value.

## Features
- `Greeting` reusable functional component
- Props: `name`, `timeOfDay`, `bgColor` (bonus)
- Conditional rendering using a config object lookup
- 4 greetings rendered for Morning, Afternoon, Evening, Night
- Dynamic accent colors, emoji watermarks, and hover animations

## How to Run

```bash
npm install
npm start
```

Then open http://localhost:3000 in your browser.
